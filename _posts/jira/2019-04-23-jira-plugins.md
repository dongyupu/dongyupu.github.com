- `Jenkins Integration for Jira Server`
  - Jira Software addon to integrate Jenkins CI.
- `Git Integration for Jira`
  - A plugin to integrate JIRA with Git. This plugin displays Git commit info in a tab on the associated JIRA issue.
- `Atlassian REST API Browser`
  - Discover, browse, and try Atlassian's rich set of REST and JSON-RPC APIs
- BigGantt
  - BigGantt plugin empowers you to dynamically manage project Work Breakdown Structure, customize filter and display the selected data the way you want it. Milestone markers, critical path, progress and custom views let you quickly evaluate your project status.
- BigTemplate
  - BigTemplate plugin empowers you to export your project data to juicy templates.
- `Confluence Issue Tab Plugin`
  - JIRA plugin that adds a Confluence tab to each issue, allowing an issue-specific Confluence page to be created or viewed. To set this up, first create an application link from JIRA to Confluence. Then, create a project link from your JIRA project to a Confluence space of your choosing. That's it! Each issue in that JIRA project will then have a "Confluence" tab from which you can create, view, or edit the Confluence page associated with that issue.
- `GitLab Listener`
  - JIRA add-on allowing the reception of GitLab push events using GitLab project web hooks.
- `JIRA Gerrit Plugin`
  - Integrates Atlassian JIRA with Gerrit code review.
- `Jira Misc Custom Fields(recommend)`
  - An assortment of Custom Fields for Atlassian Jira.
- `JSU Automation Suite for Jira Workflows`(recommend)
  - JSU is an essential add-on for every Jira customer. JSU includes additional workflow conditions, validators, post functions and custom fields.
- `Mobile for JIRA`
  - Mobile for JIRA is your ultimate mobile app on JIRA
- SAML Single Sign On for Jira
  - SAML 2.0 based Single Sign On for Jira
- `SonarQube Connector for JIRA`(recommend)
  - This plugin adds a new tab in project section to add code quality from SonarQube.
- Structure
  - Issue Hierarchy - multi-level, cross-project, hierarchical lists of issues with time aggregation, status rollup, synchronization with Links and JIRA Agile, and more.
- `Subcomponents for Jira`
  - Extend project components and versions with Subcomponents, Component Versions, Packages and more.
- `TestRail for JIRA Test Management`
  - JIRA add-on to integrate TestRail test management, QA reporting and test cases.
- Zephyr for JIRA
  - Test Management application for JIRA from Zephyr (see http://www.getzephyr.com for more details)
- xCharts - Custom Charts and Reports for Jira
  - Provides highly-customizable and lightweight Charts for Jira, as well as the possibility to script own custom charts.
- `Dynamic Forms for Jira(recommend)`
  - Show fields on screen dynamically, control user views of fields and tabs based on select field value... and a lot more.

- `Structure - Project Management at Scale`
