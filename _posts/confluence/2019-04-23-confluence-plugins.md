## Confluence plugins

- `Issue Manager for Confluence`
  - Powerful JIRA integration. Create, edit, search and embed JIRA issues in Confluence.
- `Excellentable`
  - This is a plugin for enhancing the table and search functionalities in Atlassian Confluence
- `Draw.io Confluence Plugin`
  - draw.io plugin enables creating diagrams within Confluence
- `EasyMind`
  - EasyMind allows you to create and manage mind maps directly in Confluence.
- `Comala Workflows`
  - Comala Workflows gives you full document management control in Confluence through approvals, tasks, notifications and workflows. Set customized workflows to create, review, approve and publish content - minus the complex programming languages.
- Comala Canvas
  - Comala Canvas lets you visualize and organize your Confluence content. Use boards to categorize pages visually and apply business frameworks. Use Lists to display pages into grids, providing the means to tick-off labels or set page metadata for each row.
- `Questions for Confluence Plugin`
  - Questions for Confluence
- Scroll PDF Exporter
  - The Scroll PDF Exporter exports content from Confluence pages to styled PDF documents.
- Scroll Exporter Extensions
  - The Scroll Exporter Extensions add-on contains extensions for the Scroll Exporter product family.
- SAML Single Sign On for Confluence
  - SAML 2.0 based Single Sign On for Confluence
- confluence-markdown-macro
  - This plugin provides macros to render markdown for Confluence.
- Confluence Mobile Plugin
  - This is the backend plugin for the Confluence Mobile App.
- `Bob Swift Atlassian Add-ons - Excel`
  - Your Excel data. In Confluence. Simply excellent. Display your Excel Spreadsheets in Confluence. Compatible with chart macro. Various view options. Excel file can be an attachment, accessed via URL, or a file on the Confluence file system.
- Bob Swift Atlassian Add-ons - Advanced Tables
  - Communicate information that matters. Create tables with limitless potential. Advanced Tables goes way beyond Confluence's native capabilities, transforming your data into rich, eye-catching tables.
- Bob Swift Atlassian Add-ons - Markdown
  - Use Markdown markup to create content in Confluence.
- Bob Swift Atlassian Add-ons - SQL (Pro Edition)
  - All your data. In Confluence. In real time. Display SQL query results on Confluence pages. Access JDBC compliant databases. Data feed for the chart macro. Advanced table format control. Enabled for use with Macro Security for Confluence.
- Bob Swift Atlassian Add-ons - Table Library
  - A component library enabling advanced table capabilities for other Confluence add-ons. Disabling this add-on will disable other add-on that use this component.
- Bob Swift Atlassian Add-ons - Visio
  - Display your Visio diagrams within Confluence. Uses an Internet Explorer (IE) Visio viewer control or a converted image. Converting images requires the Confluence server have a conversion package installed.
- `Atlassian REST API Browser`
  - Discover, browse, and try Atlassian's rich set of REST and JSON-RPC APIs
